Sistema para gestionar usuarios hecho con Spring Boot y Primefaces.

Sistema CRUD para gestionar usuarios, hecho con Spring Boot, JPA, Hibernate, Primefaces y Primeflex. El sistema usa Mysql como motor de base de datos, y fue hecho con la versión 17.0.8 de Java.
